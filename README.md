🧠 AI-FlashMind
全栈学习实践项目 v0.15 —— 基于 DeepSeek AI 驱动的沉浸式智能刷题工具。

🌟 项目简介
AI-FlashMind 是一个专门为高效学习者设计的 Web 刷题应用。它能够将凌乱的文本题库，通过 DeepSeek 大模型实时解析为结构化的智能练习界面，并自动生成深度解析。

本项目是我作为全栈工程师进阶之路的首个里程碑作品，专注于解决“原始题库难整理、练习无反馈”的痛点。

✨ 核心特性
🤖 AI 智能识别：支持直接上传模糊格式的 .txt 文本，AI 自动提取题目、选项及答案。

📝 动态深度解析：针对每一道题，AI 实时生成超过 50 字的专业解析，助你吃透知识点。

📱 移动端深度优化：

响应式设计：完美适配手机、平板及 PC。

交互细节：底部固定操作栏、触感反馈、沉浸式卡片布局。

⚡ 极速体验：无需后端数据库，纯前端驱动，配合 Tailwind CSS 实现丝滑流畅的视觉体验。

🚀 技术栈
前端框架: HTML5, JavaScript (ES6+)

样式处理: Tailwind CSS (CDN 加速)

AI 引擎: DeepSeek API (Chat-v3)

部署平台: GitHub Pages / Vercel
